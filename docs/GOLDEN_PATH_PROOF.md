# Golden Path Proof

This repository demonstrates one full SAR ResourceOS parcel lifecycle:

1. document number reserved and agreement issued
2. dispatch created and approved
3. gate/loading/weighbridge/release completed
4. delivery confirmed
5. reconciliation completed
6. finance handoff marked ready
7. accounting export prepared

## Demo routes
- `/`
- `/golden-path`
- `/api/demo/parcel`

## Purpose
This is the first concrete proof baseline for moving ResourceOS from architecture into live execution logic.
